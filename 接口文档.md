请求方式：POST 
接口路由：'http://localhost:8081/ai/new/session'

接口参数：
```json
{
"session_id": "b199d322-7079-4a1b-958c-1402012d75d1",
"question": "我只要数据中邮箱不为空的邮箱数据就行了"
}
```

接口响应：

```json
{
  "session_id": "b199d322-7079-4a1b-958c-1402012d75d1",
  "reply_message": "CSV下载链接\nhttp://localhost:9000/financial-exports/generated/users_2025_emails_nonempty_1765177018.csv\n\n说明\n- 仅包含邮箱非空的数据\n- 字段：Email\n- 共计：3 条"
}
```

接口说明：

1. 首次调用接口的时候，不传session_id参数。后端会自动生成，之后的同一个session的对话，使用上一次响应中的session_id
2. reply_mnessage的格式为markdown格式。
